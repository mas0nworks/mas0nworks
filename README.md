# Tencent Cloud Lighthouse

Tencent Cloud Lighthouse is a lightweight cloud server offering designed for developers, startups, and small teams.

It is built for practical cloud workloads such as websites, web applications, blogs, online stores, small programs, development environments, and testing environments. Lighthouse combines compute, storage, and networking into a simpler out-of-the-box experience, with prebuilt images, one-click application deployment, and a unified management console.

## Mission

Our mission is to make cloud deployment simpler, faster, and more approachable for developers and growing businesses.

We aim to lower the barrier to launching real applications in the cloud by providing:
- simple and streamlined setup
- ready-to-use application images and software stacks
- practical deployment paths for common workloads
- cost-efficient resource packages with clear choices

## What Lighthouse is for

Lighthouse is designed for lightweight and practical application scenarios, including:
- websites and blogs
- web apps and e-commerce apps
- mini programs and small game backends
- cloud storage and image hosting
- development, learning, and test environments
- fast deployment of popular open-source applications
